# AstroAccess — Module 1  
## Classifying Stars Using Data and Python
---
## Who is this for?
This module is designed for **everyone** who is curious about astronomy and wants to learn how astronomers actually study stars using **data**.
You do **not** need advanced mathematics.  
You only need **curiosity** and a willingness to try.

![The night sky full of stars](https://apod.nasa.gov/apod/image/2603/rapa_nui_milky_way_1024.jpeg)
*The night sky — every point of light is a star waiting to be classified.*

---
## What is this module about?
In textbooks, we often memorize star types such as:
- Main sequence  
- Giants  
- Supergiants  
- White dwarfs  

![Stellar spectral types OBAFGKM](https://apod.nasa.gov/apod/image/0105/obafgkm_noao.jpg)
*Stellar spectra across different star types — each stripe of color is a star's chemical fingerprint.*

But real astronomers do **not** memorize star types. Instead, they:
- Collect data  
- Plot graphs  
- Look for patterns  
- Draw conclusions  

In this module, **you will do the same thing using Python**.

---
## What you will learn in this module
By the end of this module, you will be able to:
- Understand what a **Hertzsprung–Russell (HR) Diagram** represents  
- Use **real numerical data** of stars  
- Plot an **HR Diagram using Python**

![Hertzsprung-Russell Diagram](https://chandra.harvard.edu/edu/formal/stellar_ev/story/hr.jpg)
*The HR Diagram — the most important graph in astronomy. You will build one yourself using Python.*
